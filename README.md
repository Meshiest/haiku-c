## C Haiku Generator

I make haiku generators for every language I learn. Here is one in C!

An example of a generated (nonsensical) haiku:

    atonality,
    worth staphylococcus owe,
    droopy blain spoils claque

### Requirements

- Make
- GCC

### Building

- `make`

### Running

- `make run` or `./haiku`
