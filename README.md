## C Haiku Generator

I make haiku generators for every language I learn. Here is one in C!

An example of a generated (nonsensical) haiku:

    adaptive withdrew 
    chic charming creek gargoyle svelte 
    childless topsoil bairn

### Requirements

- Make
- GCC

### Building

- `make`

### Running

- `make run` or `./haiku`
